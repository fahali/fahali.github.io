Trivia Night

A trivia game serving questions from the Open Trivia Database, a collection spanning 11,000+ questions. Players can customize their experience by choosing between 24 question categories, 3 difficulty levels, 2 types of questions, timed scoring, and more. It is built completely in vanilla HTML, CSS, and Javascript, and all interactions are handled via DOM manipulation. Session tokens are used to continously serve "fresh" questions within a 6 hour timeframe.

You can check out the code on my GitHub: Trivia Night

Times Say

A single page application that presents movie reviews from The New York Times archives, a collection spanning 29,000+ articles, via The New York Times Movie Review API. For an attractive user interface, attaches review detail components to movie posters fetched from The Movie Database API. Built using React and React Bootstrap. A highlight of this project is the pagination feature, allowing the user to page through the entire dataset, amounting to more than 1450 pages.

You can check out the code on my GitHub: Times Say

Better Bus Time

A love letter to the best city in the world, Better Bus Time aims to improve upon MTA Bus Time. New Yorkers can pin their favorite bus routes and bus stops to the front page, so the transit information most pertinent to them is only a click away. The interface is designed to be easy to use so even visitors can easily find information on routes and stops relevant to their travels. The client side application is built in React. The server side application is built in Express, talking to a MongoDB cluster on Mongo Atlas, via Mongoose. The backend also serves as a proxy, passing requests and responses between the frontend and the two MTA APIs: the OneBusAway "Discovery" API, and the SIRI API.
